# 关键知识点总结

- CSS Reset：使用通配符选择器重置默认样式，确保跨浏览器一致性
- Flexbox 布局：用于实现居中和响应式布局
- Position 定位：
  - relative：为子元素提供定位参考
  - absolute：实现动画和重叠效果
- 过渡动画：
  - transition 属性实现平滑动画
  - transform 实现位移效果
- 响应式设计：
  - 使用媒体查询(@media)
  - 使用相对单位(vh, vw)
  - 针对不同屏幕大小优化布局
- CSS 优化技巧：
  - 使用 box-shadow 增加立体感
  - 使用 overflow:hidden 控制内容溢出
  - 使用伪元素(::before)实现复杂动画
- 交互设计：
  - hover 效果
  - 动画过渡
  - 视觉反馈
